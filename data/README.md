# Data Directory

This directory contains read-only scenario data for the assessment modules. **Do not modify these files.**

| File | Used By | Description |
|------|---------|-------------|
| `aws_cost_report.json` | Module 1 | AWS Cost Explorer export for cost optimization analysis |
| `site_spec.json` | Module 3 | Customer site specification for network planning |
| `onvif_mock_response.xml` | Module 3 | Mock ONVIF camera discovery response |
| `incident/scenario_1/` | Module 2 | CrashLoopBackOff incident data |
| `incident/scenario_2/` | Module 2 | Service unreachable incident data |
| `incident/scenario_3/` | Module 2 | Rollout stuck incident data |
| `debug_scenario/` | Module 5 | Edge-to-cloud debugging scenario data |
